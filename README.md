# TERRACIM APP

## Description
TERRACIM APP est une application Java développée en travail de groupe.
Elle a pour objectif de faciliter la communication et la coordination entre
les différents acteurs d’un projet de construction afin d’assurer sa réalisation complète.

L’application centralise les échanges entre les clients et les employés
et permet un suivi efficace des projets du bâtiment.

## Objectif du projet
- Faciliter la communication entre les clients et les employés
- Assurer le suivi et la gestion des projets de construction
- Améliorer l’organisation du travail dans le domaine du bâtiment
- Appliquer les concepts de la programmation orientée objet en Java
- Travailler en équipe sur un projet informatique

## Technologies utilisées
- Java
- JavaFX /MVC /MAVEN
- MySQL 
- Git & GitHub

## Espaces utilisateurs
L’application contient quatre espaces principaux :

- **Espace Directeur**  
  - Supervision générale des projets  
  - Consultation de l’avancement global  
  - Prise de décisions et validation  

- **Espace Chef de chantier**  
  - Suivi des travaux sur le chantier  
  - Mise à jour de l’état des projets  
  - Communication avec le directeur et la secrétaire  

- **Espace Client**  
  - Consultation de l’état d’avancement du projet  
  - Communication avec l’équipe  
  - Réception des informations et notifications  

- **Espace Secrétaire**  
  - Gestion administrative  
  - Organisation des informations  
  - Communication entre les différents acteurs  

## Fonctionnalités
- Authentification des utilisateurs
- Gestion des projets de construction
- Communication entre clients et employés
- Suivi de l’avancement des projets
- Interface graphique simple et intuitive

## Installation
1. Cloner le projet depuis GitHub
2. Ouvrir le projet dans un IDE (IntelliJ)
3. Configurer la base de données 
4. Lancer l’application

## Travail en groupe
Ce projet a été réalisé en travail de groupe dans un cadre académique.
Chaque membre du groupe a participé à la conception et au développement de l’application.
